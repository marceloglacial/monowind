## New App
- Create pagkage on `app`
- Install and congigure `tailwind`
- Add `"../../packages/styles-library/**/*.{js,ts}",` to tailwind content
- Add `@monowind/styles-library` to your project
- Run `yarn install`
- Run/start your project
